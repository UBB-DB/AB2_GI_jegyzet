Tárolt eljárások#

CREATE PROCEDURE <schema_name>.<procedure_name> @<parameter_name> = ‘VALUE’ <OUTPUT> AS … ALTER PROCEDURE <schema_name>.<procedure_name> AS …
EXEC <schema_name>.<procedure_name> @<parameter_name> = ‘VALUE’

  • Visszatérési értékkel
    CREATE PROCEDURE HumanResources.uspGetEmployeesTest2 @LastName nvarchar(50), @FirstName nvarchar(50) AS … RETURN(kod) Hivás: DECLARE @result int, @out_param int; EXECUTE @result = my_proc @out_param ;

  • Ki- és bemeneti paraméterekkel valamint alapértelmezett értékekkel és visszatérési értékkel CREATE PROCEDURE Sales.usp_GetSalesYTD @SalesPerson nvarchar(50) = NULL, @SalesYTD money = NULL OUTPUT Hivás:   DECLARE @SalesYTDForSalesPerson money, @ret_code int 
    EXECUTE @ret_code = Sales.usp_GetSalesYTD ’Blythe’, @SalesYTD = @SalesYTDForSalesPerson OUTPUT;

Feladatok:#

- Irjunk tárolt eljárást mely 1-et térit vissza ha van egy termék volt rendelve különben 0-t, bemenetként pedig megkapja a termék nevét.

- Bővitsük az előzőt azzal, hogy visszatériti kimeneti paraméterként az illeto termék rendelési mennyiségét.

CREATE OR ALTER PROCEDURE spRaktarMennyiseg 
    @termekNev nvarchar(50),
    @mennyiseg int OUTPUT
AS
BEGIN
    SET NOCOUNT ON;
    --DECLARE @rendelesSzam int =
    SET @mennyiseg = (SELECT COUNT(SalesOrderId) FROM SalesLT.SalesOrderDetail
        WHERE ProductId IN (SELECT ProductId FROM SalesLT.Product WHERE Name LIKE @termekNev)) 
    IF @mennyiseg = 0
        RETURN 0
    ELSE
        RETURN 1
END

DECLARE @termekNeve nvarchar(50) = '%Bike%'
DECLARE @returnCode int, @raktarMennyiseg int
EXECUTE @returnCode= spRaktarMennyiseg @termekNeve , @mennyiseg=@raktarMennyiseg OUTPUT
SELECT @returnCode, @raktarMennyiseg
